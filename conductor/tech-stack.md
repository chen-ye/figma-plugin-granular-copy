# Tech Stack

## Core Technologies
- **Language:** TypeScript (TS) for strong typing and developer productivity.
- **Frontend Framework:** React for building a modular and reactive user interface for the persistent "Paste..." window.

## Development Tools
- **Build Tool/Bundler:** Vite for fast HMR (Hot Module Replacement) and efficient production builds.
- **Testing Framework:** Vitest for unit and integration testing, leveraging its native Vite integration.

## Figma Specifics
- **Figma Plugin API:** Target the latest stable version of the `@figma/plugin-typings`.
- **UI Kit:** Utilize `@figma-plugins/ui` or a custom implementation of Figma's internal design system to ensure the "Native Aesthetic" guideline is met.
